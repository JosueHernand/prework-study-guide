HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It's important to add semantic HTML elements so other developers and the browser can read it.
  • The HTML attributes are always defined in the beginning never at the closing element.
  • The meta element defines metadata such as the character set, description, keyword, author, and viewpoint.
  • Metadata is a set of data that provides information about other data

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Do Not Repeat Yourself or DRY helps developers reuse or share pieces of code to reduce the number of lines.
  • We can combine two selectors into a single CSS rule by using a comma if you're going to be using the same styling.
  • A link element creates the connection to the external file.
  • The section element helps organize the content on the page.
  • The Declaration part contains two important components which is the property and the value.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • To create a directory we use the command mkdir name.
  • To make a new file we use the command touch name.
  • Before you push a code in the feature branch you want to make sure you're on the right branch by checking the status.
  • When on the Github make sure the branch main and the comparison branch is on the one you want to make create a pull request on.
  • It is always important that your partners take a look before merging the branches.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The for loop has 3 important statement which is the starting point of the loop, the confition, and the one that allows the array to iterate over each item.
  • JavaScript is a programming language that allows users to interact with websites.
  • Whenever you work with Java every code ends with a semicolon.
  • An if statement is an introduction decision into the coding only executes whenever it meets the condition.
  • Adding three equal signs means that it will check to see if two values are equal and will return with either a true or false.
  • An array is a single variable that is sed to hold a group of data. It is useful for storing multiple items with different data.
